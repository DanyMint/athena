# Athena
Athena is a small application to keep track of applicants.

## 🚀 Installation

⚠️ Pre-requisites

    Ensure Docker Engine is installed and running.

Clone this repo
~~~sh
git clone https://github.com/DanyMint/athena.git
~~~

Go into direcoty
~~~sh
cd ./athena
~~~

add permissions `install.sh`
~~~sh
sudo chmod +x ./install.sh
~~~

Run installation
~~~sh
sudo ./install.sh
~~~
